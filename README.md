# Portfolio Personnel - Projet 6.2 

## 📌 Description

Ce projet est un portfolio personnel réalisé dans le cadre de ma formation en développement web à Akieni Academy.

L'objectif était de concevoir une interface web moderne et responsive uniquement avec **HTML5 et CSS3**, afin de présenter mon profil, mes compétences, mon parcours et mes projets.

Ce portfolio constitue ma première vitrine professionnelle en ligne et servira de base pour intégrer progressivement de nouvelles technologies et fonctionnalités au fil de mon évolution en développement web.

## Fonctionnalités

- Présentation personnelle
